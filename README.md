# Proyecto_Katas_Python
Proyecto Katas PYTHON
Fecha finalización 17/04/2026

1.	DESCRIPCIÓN

A partir del fichero “EnunciadoDataProjectPython.pdf” que contiene los enunciados de los ejercicios propuestos (41) Se crea todo el proyecto en Viual Studio, inicialmente en un fichero .ipynb que al final se exporta a formato .py

2.	ESTRUCTURA

El proyecto consta de 3 ficheros:

a)	EnunciadoDataProjectPython.pdf: Es el PDF descargado que contiene todos los enunciados 
  
b)	Proyecto_Katas_Python.ipynb: Fichero original en el que se ha hecho todo el desarrollo. 
    Cada ejercicio está encabezado con un Markdown con su número y enunciado y a continuación una celda de código con la solución propuesta. Se incluyen algunos comentarios

c)	Proyecto_Katas_Python_SMC.py: Exportación del fichero anterior en formato .py 

3.	INSTALACIÓN Y REQUISITOS

El proyecto se ha realizado utilizando las aplicaciones:

-VISUAL STUDIO CODE / JUPYTER NOTEBOOK

-GITHUB para publicar el proyecto

Para ejecutarlo solo necesitas tener instalado PYTHON
